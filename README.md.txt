# Smart Water Monitoring System

An IoT-based Smart Water Monitoring and Control System using ESP32, MQTT, BLE, HTML, CSS, JavaScript, and Python.

## Features
- ESP32 MQTT Dashboard
- Fan Control (Red LED)
- Light Control (Yellow LED)
- Pump Control (Green LED)
- Tank Water Level Monitoring
- Online/Offline Status
- BLE WiFi Setup
- Change WiFi from Dashboard
- Activity Logs with Timestamps
- CSV Download
- Responsive Dashboard
- Python MQTT Controller

## Technologies Used
- ESP32
- MQTT
- HTML
- CSS
- JavaScript
- Python
- BLE
- GitHub Pages
